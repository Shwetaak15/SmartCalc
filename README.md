 🧮 SmartCalc

SmartCalc is a modern and responsive calculator built using HTML, CSS, and JavaScript. It offers a clean interface and smooth user experience for performing everyday calculations.

 ✨ Features

* Responsive design
* Basic arithmetic operations
* Calculation history
* Keyboard support
* Clear and Backspace functionality
* User-friendly interface

 🚀 Technologies Used

* HTML5
* CSS3
* JavaScript

 📂 Project Structure

```text
SmartCalc/
│
├── index.html
├── style.css
└── README.md
```

🛠️ How to Run

1. Download or clone the repository.
2. Open `index.html` in your browser.
3. Start calculating!

🎯 Future Improvements

* Scientific calculator mode
* Dark/Light theme toggle
* Advanced history management
* Custom themes
